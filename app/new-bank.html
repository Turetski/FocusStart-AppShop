Новый ЦФТ-Банк

09 сентября 2016

Приложение лицензировано в 25 банках
Тип приложения: базовое
Разработчик: ЦФТ
Код по каталогу: 1B365001

Требования: совместимо с v.14.1, ТЯ v.7.4.0.0 и выше при наличии лицензии на проигрыватель форм в web

В корзину

Основные функции

  • Выполнение операций по банковским продуктам, используемым клиентом, и оформление новых продуктов банка непосредственно из Единого окна
  • Возможность определения одного банковского продукта для отображения в Едином окне как рекомендуемого (для формирования списка адресно рекомендуемых продуктов необходима лицензия на Приложение Управление целевым маркетингом)
  • Начисление баллов операционным сотрудникам за выполнение операций по банковским продуктам и отображение результатов в Едином окне (для использования гибкой настройки условий начисления баллов необходима лицензия на Приложение Игрофикация)
